# Muv Construction - Design Style Guide

## Design Philosophy

### Visual Language
**Modern Industrial Elegance**: Combining the raw strength of construction with sophisticated design aesthetics. The visual language emphasizes precision, reliability, and innovation while maintaining approachability and trust.

### Color Palette
**Primary Colors**:
- **Deep Steel Blue** (#1e3a8a) - Primary brand color, representing trust and stability
- **Charcoal Gray** (#374151) - Secondary color for text and accents
- **Warm White** (#fefefe) - Background and negative space

**Accent Colors**:
- **Copper Orange** (#ea580c) - Call-to-action elements and highlights
- **Sage Green** (#16a34a) - Success states and environmental focus

### Typography
**Primary Font**: "Inter" - Clean, modern sans-serif for headings and UI elements
**Secondary Font**: "Source Sans Pro" - Readable sans-serif for body text and descriptions
**Accent Font**: "Playfair Display" - Elegant serif for special headings and branding

### Visual Hierarchy
- **H1**: 3.5rem, bold, Deep Steel Blue
- **H2**: 2.5rem, semibold, Charcoal Gray
- **H3**: 1.875rem, medium, Deep Steel Blue
- **Body**: 1rem, regular, Charcoal Gray
- **Caption**: 0.875rem, medium, muted

## Visual Effects & Styling

### Hero Section Design
- **Background**: Cinematic construction site imagery with subtle parallax
- **Logo Treatment**: Animated blue capital "M" with geometric construction elements
- **Text Animation**: Typewriter effect for company name with color cycling emphasis
- **Layout**: Split-screen design with hero imagery and content overlay

### Interactive Elements
- **Buttons**: Rounded corners, subtle shadows, hover lift effects
- **Cards**: Clean borders, hover animations with 3D tilt
- **Forms**: Floating labels, real-time validation, smooth transitions
- **Navigation**: Sticky header with blur background effect

### Animation Library Usage
- **Anime.js**: Smooth page transitions, element reveals, and micro-interactions
- **Matter.js**: Physics-based hero section with floating construction elements
- **ECharts.js**: Service calculator visualizations with animated data
- **Splide.js**: Project portfolio carousels with Ken Burns effect

### Background Effects
- **Primary**: Subtle geometric pattern overlay with construction blueprint aesthetics
- **Secondary**: Gradient mesh with industrial color palette
- **Accent**: Particle system representing construction dust and materials

### Scroll Motion
- **Reveal Animations**: Elements fade in with 20px upward motion
- **Parallax**: Background elements move at 0.5x scroll speed
- **Stagger Effects**: Sequential animation delays for card grids
- **Trigger Points**: Animations activate when elements enter top 60% of viewport

### Hover Effects
- **Cards**: 3D tilt with depth shadow expansion
- **Images**: Zoom with gradient mask overlay
- **Buttons**: Color morphing with glow edges
- **Links**: Animated underlines with color transitions

### Data Visualization Style
- **Color Scheme**: Monochromatic blue variations with copper accents
- **Chart Types**: Clean bar charts, progress rings, and timeline visualizations
- **Animation**: Smooth data entry with staggered reveals
- **Interaction**: Hover tooltips with project details

### Mobile Responsiveness
- **Breakpoints**: 640px, 768px, 1024px, 1280px
- **Typography Scale**: Responsive font sizing with fluid scaling
- **Touch Targets**: Minimum 44px for all interactive elements
- **Navigation**: Collapsible mobile menu with smooth animations

### Accessibility Features
- **Contrast Ratios**: Minimum 4.5:1 for all text
- **Focus States**: Clear keyboard navigation indicators
- **Alt Text**: Descriptive image alternatives
- **Screen Reader**: Semantic HTML structure with ARIA labels
